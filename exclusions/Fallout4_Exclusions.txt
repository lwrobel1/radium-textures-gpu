# Fallout 4 Exclusions
# Based on VRAMr analysis and texture format best practices
#
# Pattern types:
#   *pattern.dds  - Wildcard filename match
#   \path         - Directory exclusion
#   filename.dds  - Exact filename match

# ===== PBR Texture Suffixes =====
# These use specific channel layouts that shouldn't be modified
*_color.dds
*_emissive.dds
*_normal.dds
*_rmaos.dds
*_p.dds

# ===== Specular/Glow/Environment Maps =====
# Specular is typically embedded in normal map alpha channel
# Note: *_s.dds won't match *_rmaos.dds (different suffix)
*_s.dds
*_g.dds
*_e.dds

# ===== Other Texture Suffixes =====
*_OpenGL.dds

# ===== LOD Textures =====
# Already optimized for distance, resizing breaks LOD system
*lod*_p.dds
*_lod.dds
