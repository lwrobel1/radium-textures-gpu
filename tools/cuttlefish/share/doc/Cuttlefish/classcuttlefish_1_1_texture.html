Detailed Description

Class describing a texture.

A texture is created from one or more images, and are then converted to the final texture format. From that point, it can be saved to a texture file.

◆ adjustImageValueRange()

Type  type,
Image::Format  origImageFormat = Image::Format::Invalid 
)
static

Adjusts the value range for an image.

This will adjust values based on reasonable expectations from the original image format to the texture type. The current situations considered are:

  • Input images with a UNorm value range will be converted to [-1, 1] range when used with SNorm type. This will convert to a float type if not already one.
  • Input images with a UNorm value range will be converted to the appropriate integer range.
Parameters
[in,out]imageThe image to adjust the values for.
typeThe target texture type the image will be converted to.
origImageFormatThe original image format before any conversions for internal processing. If set to Invalid, the image's current format will be used.

◆ generateMipmaps()

bool generateMipmaps ( Image::ResizeFilter  filter = Image::ResizeFilter::CatmullRom,
unsigned int  mipLevels = allMipLevels,
const CustomMipImages &  customMipImages = {} 
)

Generates mipmaps for the texture.

All of the images fro the first mip level must be set beforehand. All other existing levels will be ignored.

To customize images used for specific mip levels, a mapping from image index to the custom image may be provided via customMipImages. Any images will be resized to the appropriate dimensions, and may be set to either use for that one mip image or to continue using for following images in the mip chain.

Remarks
When using custom mip images for 3D textures, when providing custom image for a given mip level all depth images must be provided, and they must all use the same replacement value.
Parameters
filterThe filter to use for resizing.
mipLevelsThe number of mipmap levels to generate.
customMipImagesCustom images to use for individual mip levels.
Returns
False if the texture isn't valid.
